com.upokecenter.text.ICharacterEncoding

# com.upokecenter.text.ICharacterEncoding

public interface ICharacterEncoding

Defines methods that can be implemented by classes that convert to and from bytes and character code points.

Methods

  • ICharacterDecoder GetDecoder()
    Creates a decoder for this character encoding with initial state.

  • ICharacterEncoder GetEncoder()
    Creates an encoder for this character encoding with initial state.

Method Details

GetEncoder

ICharacterEncoder GetEncoder() Creates an encoder for this character encoding with initial state. If the  encoder is stateless, multiple calls of this method can return the same  encoder.

Returns:

  • A character encoder object.

GetDecoder

ICharacterDecoder GetDecoder() Creates a decoder for this character encoding with initial state. If the  decoder is stateless, multiple calls of this method can return the same  decoder.

Returns:

  • A character decoder object.
